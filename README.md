# My Portfolio 💼

## Features ✨
This was one of my first solo projects, and my main goals were:
- **Responsive Design** 🌐 (e.g., hover over my pixelated self, created with Dall-E, or interact with the Spline.js objects I made for Wavelength and GARDE)
- **Blog Section** 📝 to share my thoughts and experiences
- **Personalized Showcase** 🎓 to present my education and experience in my own unique way

## Technologies Used 💻
While the code is mostly basic HTML/CSS, I plan to revisit and streamline it in the future.
- HTML
- CSS
- JavaScript
- Node.js

## Running Locally 🚀

To run the website locally and customize it, you need to have Node.js installed. Once you have Node.js installed, follow these steps:

1. Clone the repository.
2. Navigate to the project directory.
3. Install the dependencies using `npm install`.
4. Start the development server using `npm start`.

The website should now be running on `localhost:8080`.
